Tema escolhido:
Escolhi desenvolver um site utilizando a empresa do meu irmão como inspiração, aproveitando que ele desenvolve imagens 3d para desenvolver esse site

Link: https://luizgustavos07.github.io/AP2-NEX/index.html

Estrutura das Páginas
Eu desenvolvi o site dividido em 3 paginas html, sendo elas:
index.html (página principal): Página principal com carrossel de imagens e texto de apresentação.
portfolio.html (portfólio): Mostrando alguns dos projetos que eles desenvolveram através dos cards
servicos.html (serviços): Listei algumas areas de atuação que a empresa trabalha

Componentes Bootstrap Utilizados

Navbar & Dropdown: menu superior com interação e o dropdown em serviços
Carousel:Banner deslizante automático na pagina inicial.
Cards: Cards para organizar as imagens do portfolio
Toast: Utilizei o toast para mostrar aonde entrar em contato para tirar dúvidas etc
Utilizei bastante o py- e my- para dar os espaçamentos
Alem de usar o background color (bg), cores de borda (border) rounded-0 para as bordas ficarem retas, w-100 para acompanhar o tamanho, opacity-50 para mudar a opacidade da imagem
Usei támbem o object-cover-fit para tentar não deixar as imagens "achatadas" no telefone.

Design e Layout:
Decidi realizar um site mais minimalista, para que fique um pouco mais sofisticado, já que se trata de um studio de arquitetura 3d, então as cores predominantes sao neutras, como cinza, branco e preto. Trazendo uma sofisticação

Utilização display-flex:
Utilizei o d-flex, junto de flex-column e min-vh-100 para deixar cada parte do site responsivo e alinhado, alem de ter que colocar o flex-grow-1 para que a parte inferior ficasse na parte inferior da tela

Observaçoes relevantes sobre o desenvolvimento:
Passei um pouco de trabalho para jogar a barra de baixo para o fundo da página, tive que recorrer a um video no youtube que ensinava sobre o flex-grow e o object-cover-fit, alem de sofrer um pouco támbem com o Toast, por conta do do script que acabou nao funcionando de inicio
Mas esse trabalho foi fundamental para eu me aprofundar ainda mais em bootstrap e entender o quanto essa ferramenta facilita o desenvolvimento das páginas web diariamente
